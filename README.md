# Ecommerce-Frontend

A premium, modern, and high-performance e-commerce responsive web application built with **Angular**. This project focuses on delivering an exceptional user experience (UX) inspired by modern clean layouts, featuring advanced state management and secure client-side workflows.

---

## ✨ Key Features

*   **Modern & Premium UI/UX:** Designed with a focus on clean typography, responsive grid structures, and an Airbnb-style elegant layout.
*   **Advanced State Management:** Utilizes **Angular Signals** for reactive, lightweight, and high-performance state/data management.
*   **Fully Responsive:** Optimized for all screen sizes (Desktop, Tablet, Mobile) using modern CSS methodologies.
*   **Secure Architecture:** Implements robust client-side routing, guards, and secure token/session handling practices.

## 🛠️ Tech Stack & Architecture

*   **Framework:** Angular (v17+)
*   **State Management:** Angular Signals / RxJS
*   **Styling:** Advanced CSS / SCSS (Flexbox, Grid, Custom Properties)
*   **Features:** Component-driven architecture, Standalone Components, Modular Services

---

## 🚀 Getting Started

To run this project locally, follow these steps:

1. **Clone the repository:**
   git clone https://github.com/yemresener/Ecommerce-Frontend

2. **Install dependencies:**
   npm install

3. **Run the development server:**
   ng serve
